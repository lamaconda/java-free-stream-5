1) Объект - дверь
	Состояние:
		- есть / нет
		- открыта / закрыта
	Поведение:
		- открывается - в помещение можно попасть
		- закрывается - вход в помещение закрыт

2) Объект - машина
    Состояние:
        - исправна / не исправна
        - движется / не движется
    Поведение:
        - едет / останавливается

3) Объект - телефон
    Состояние:
        - включен / выключен
        - смартфон / кнопочный
    Поведение:
        - звонит
        - воспроизводит аудио/видео

4) Объект - утка
    Состояние:
        - количество лап
        - цвет
    Поведение:
        - крякает / молчит
        - летит / плывет

5) Объект - велосипед
    Состояние:
        - количество колес
        - количество сидений
    Поведение:
        - движется

6) Объект - ручка
    Состояние:
        - цвет корпуса
        - тип стержня
    Поведение:
        - пишет

7) Объект - компьюетрная мышь
    Состояние:
        - включена / выключена
        - шариковая / лазерная
    Поведение:
        - движется в какую-либо сторону
        - нажимается какая-либо кнопка

8) Объект - компьютер
    Состояние:
        - включен / выключен
        - исправен / не исправен
    Поведение:
        - включается / выключается
        - воспроизводит видео

9) Объект - цветок
    Состояние:
        - цветущий / не цветущий
        - количество цветов
    Поведение:
        - вянет
        - распускается

10) Объект - настольная лампа
    Состояние:
        - включена
        - выключена
        - тип лампочки
    Поведение:
        - включение
        - выключение
